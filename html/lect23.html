Working with Hyperlinks

<a href="url">: Anchor tag for creating hyperlinks

Link Examples:

  • Absolute URL: <a href="https://www.google.com" target="_blank">Google</a>
  • Relative Link: <a href="assignments/project1.html">Project 1</a>
  • Email Link: <a href="mailto:student@university.edu">Email Me</a>
  • Document Link: <a href="files/syllabus.pdf">Download Syllabus</a>
  • Page Section: <a href="#resources">Jump to Resources</a>
